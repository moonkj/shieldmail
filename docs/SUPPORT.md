# 지원 — ShieldMail

## 자주 묻는 질문

### 방패 아이콘이 안 보여요
1. Safari 설정 > 확장 프로그램 > ShieldMail이 켜져 있는지 확인
2. "모든 웹사이트 허용"이 선택되었는지 확인
3. HTTPS 사이트에서만 동작합니다 (HTTP 미지원)

### 인증 코드가 안 와요
- 이메일 발송에 수 초~수십 초 소요될 수 있습니다
- 방패 버튼의 파란 펄스 표시가 대기 중을 의미합니다
- 코드 수신 시 화면 우상단에 토스트로 표시됩니다

### 무료 한도를 초과했어요
- 무료 플랜은 하루 1회 생성 가능합니다
- UTC 자정(한국 오전 9시) 기준으로 리셋됩니다
- Pro 구독 시 하루 20회까지 사용 가능합니다

### 구독을 해지하고 싶어요
- 설정 > Apple ID > 구독에서 ShieldMail 구독 관리
- 또는 앱 내 설정 > 구독 상태 > Apple 구독 관리

### 환불받고 싶어요
- Apple App Store의 환불 정책에 따릅니다
- https://reportaproblem.apple.com 에서 신청 가능
