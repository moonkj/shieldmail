1. 서비스 개요

ShieldMail은 임시 이메일 주소를 생성하여 웹사이트 가입 시 개인 이메일을 보호하는 Safari 확장 프로그램입니다.

2. 이용 조건

무료 플랜: 하루 1회 임시 이메일 생성. 이메일 주소 1시간 후 자동 만료. OTP/링크 10분 후 삭제.

Pro 플랜 (월 $0.99): 하루 20회 생성. Apple iTunes 계정 결제. 월간 자동 갱신. 만료 24시간 전 갱신. 설정에서 언제든 해지 가능.

3. 금지 사항

  • 스팸, 피싱, 사기 목적의 사용
  • 자동화 도구를 통한 대량 생성
  • 타인의 계정 접근 시도
  • 서비스 보안 우회 시도

4. 면책 조항

  • ShieldMail은 임시 이메일 서비스이며 이메일 전달을 보장하지 않습니다
  • 서비스 중단, 데이터 손실에 대해 책임지지 않습니다
  • 사용자는 서비스를 합법적 목적으로만 사용해야 합니다

5. 구독 및 환불

  • 구독은 Apple App Store를 통해 처리됩니다
  • 환불은 Apple의 환불 정책에 따릅니다
  • 해지 시 현재 기간 종료까지 Pro 기능 사용 가능

6. 개인정보

7. 약관 변경
